Connect Device

Click Connect, then place your Daisy into BOOTLOADER MODE and select it from the USB popup.

. To enter BOOTLOADER MODE, simply press CONNECT, and then RESET on the Daisy Seed. "DAISY BOOTLOADER" should appear in the popup window.

. If "DAISY BOOTLOADER" disappears from the popup after only a few seconds, simply press BOOT (ONCE) while "DAISY BOOTLOADER" is displayed in the popup window. This will lock in Bootloader mode indefinitely.

. Once done correctly, a solid Red LED will be above the RESET button, and a pulsing LED should be above BOOT.
