Troubleshooting

🛑 While users have reported intermittent success with Firefox and Safari, Google Chrome is the only officially supported browser for flashing firmware to Daisy.

Reset USB Driver with Zadig (Windows Only)

If Chrome cannot find your device, you may need to install the correct USB driver:

  1. Download Zadig (latest version).
  2. Launch Zadig.
  3. Put device into BOOTLOADER mode by following the above steps.
  4. In Zadig, click Options then List All Devices.
  5. Select Daisy Bootloader from the dropdown.
  6. Set the target driver (right side) to WinUSB.
  7. Click Replace Driver.

After Zadig finishes, return to Chrome and try connecting again.
